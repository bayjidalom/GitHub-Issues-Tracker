### 1. What is the difference between var, let, and const?

Answer:

var -> Old JavaScript variable declaration. It is function scoped and can be re-declared and updated.

let -> Introduced in ES6. It is block scoped. The value can be updated, but it cannot be re-declared in the same scope.

const -> Also block scoped. Once declared, the value cannot be reassigned.


### 2. What is the spread operator (...)?

Answer:

The spread operator (...) is used to expand or copy elements of an array or object.


### 3. What is the difference between map(), filter(), and forEach()?

Answer:

map() -> Loops through an array and returns a new array after modifying each element.

filter() -> Returns a new array containing elements that match a condition.

forEach() -> Loops through the array but does not return a new array.



### 4. What is an arrow function?

Answer:

An arrow function is a shorter syntax for writing functions, introduced in ES6.


### 5. What are template literals?

Answer:

Template literals are a way to write strings using backticks ( ).
They allow embedding variables using ${} inside the string.
